#420Chan
Foro basado en el diseño de tablones de imágenes como 4Chan o Futaba
#Funcionalidad
- Posibilidad de registrarse como usuario con imagen de perfil.
- Navegación por el feed con opciones de filtrado por usuario y por fecha.
- Creación de publicaciones con imágenes.
- Creación de comentarios sobre las publicaciones.
- Página de perfil donde se puede cambiar la contraseña o la imagen de perfil.
